# GatherUp UI 概念稿 · v2「安静的美术馆」

> 本轮依据用户提供的博物馆 App 参考图重做：奶油白底、柔和粉彩卡片、优雅衬线标题、单线条插画点缀、精致留白。v1 的三个方向（A 清新活力 / B 极简高级 / C 温暖社区）已被否定并废弃。
>
> 本目录为纯设计产物，不影响 `src/` 产品代码。HTML 均为独立静态文件（内联 CSS），双击浏览器打开即可，建议先用手机宽度（390px）预览。

## 页面

| 文件 | 内容 |
|---|---|
| [v2-home.html](./v2-home.html) | 首页（活动广场）：标题栏、下划线式 tab、5 张粉彩活动卡片（取自 `src/lib/mock-data.ts` 真实活动）、底部胶囊导航 |
| [v2-detail.html](./v2-detail.html) | 活动详情：大衬线标题、信息卡、票档、悬浮报名 CTA 条 |

两页顶部有互切链接。移动端优先（390px 完美呈现），720px+ 居中限宽。

## 配色板

| 角色 | 色值 | 用途 |
|---|---|---|
| 背景 | `#F7F5F0` | 全页底色（暖调米白，绝不用纯白） |
| 墨色 | `#3A362F` | 主文字 |
| 次级墨 | `#7A756A` | 辅助文字 |
| 淡墨 | `#A39D8F` | 标签 / 占位 |
| 灰薰衣草紫 | `#E3E0EA` | 粉彩卡片 1（雾感低饱和） |
| 鼠尾草灰绿 | `#E0E3DB` | 粉彩卡片 2（雾感低饱和） |
| 暖沙色 | `#EBE6DC` | 粉彩卡片 3（雾感低饱和） |
| 陶土铜 | `#A97A5B` | 仅用于圆形 CTA 按钮与关键强调（全页唯一深色点缀，柔化后仍可辨识） |
| 深炭 | `#2E2B26` | 底部胶囊导航 / CTA 条底色 |

粉彩卡片按 紫 → 绿 → 沙 → 紫 → 绿 的节奏轮换，全稿不超过 3 种粉彩。

## 字阶表（严格执行，无随意值）

| 层级 | 字号 | 字体 | 字重 | 用途 |
|---|---|---|---|---|
| display | 32px | Noto Serif SC | 600 | 品牌名 / 详情页大标题（letter-spacing -0.02em） |
| title | 22px | Noto Serif SC | 600 | 卡片标题 / 价格数字 |
| body | 15px | Noto Sans SC（日期、价格用 Serif） | 400–500 | 正文、tab、信息主行（行高 1.6） |
| caption | 12px | Noto Sans SC | 400 | 城市标签、描述、辅助信息（标签类 letter-spacing 0.16em） |

中文标题字重不超过 600，避免过粗；日期与价格数字改用衬线体增加质感。

## 间距表（8pt 网格）

| 数值 | 用途 |
|---|---|
| 8px | 元素内小间距（标题与副题之间、导航项间距） |
| 16px | 卡片之间、信息行之间 |
| 24px | 页面左右留白（390px 视口）、卡片内边距（≥24）、tab 列间距、悬浮导航离底距离 |
| 40–48px | 区块之间、页首上留白 |

## 布局骨架（学自参考图）

- 顶部：简洁标题栏（衬线品牌名 + 1px 细描边圆形图标按钮）
- 分段 tab：下划线式，激活项 1.5px 墨色下划线（参考图 Current/Upcoming 样式）
- 活动卡片：大粉彩色块、圆角 24px、无边框无阴影；左侧文字（衬线标题 + 衬线日期 + 一句话描述），右上角单线条插画，右下角 40px 陶土色圆形箭头按钮
- 底部：深炭色胶囊药丸导航（首页）；详情页为同款胶囊 CTA 条（价格 + 56px 陶土圆钮）
- 装饰：页面背景与每张卡片各一处 line art（1px 描边、低透明度），克制使用
